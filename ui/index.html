[MOBACON]

Команда проекта выполнило колоссальную работу по проработке фирменного стиля!

[GameDude]

В рамках проекта [GameDude] планируется создать уникальную площадку для взаимодействия игроков.

Команда проекта выполнило колоссальную работу по построение уникальной архитектуры базы данных, а так же адаптивности данного сайта.
